Como publicar no GitHub Pages

1. Criar o repositório

Acesse github.com e faça login
Clique em New repository (botão verde no canto superior direito)
Dê um nome ao repositório — ex: maestria-atendimento
Deixe como Public
Clique em Create repository


2. Fazer upload do arquivo


Na página do repositório recém-criado, clique em uploading an existing file
Arraste o arquivo index.html para a área indicada
Clique em Commit changes


3. Ativar o GitHub Pages


No repositório, clique em Settings (aba no topo)
No menu lateral esquerdo, clique em Pages
Em Branch, selecione main e deixe a pasta como / (root)
Clique em Save
